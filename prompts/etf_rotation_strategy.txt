## Objective

Develop a Python script that implements an ETF rotation strategy using local bhavcopy data.


## Strategy Logic
1. Data Extraction
	- Load the most recent 350 trading days of data based on today’s date.
	- Extract data only for the specified ETF symbols (etf_list).
2. Trend Filter (200-day SMA)
	- Calculate the 200-day Simple Moving Average (SMA) for each ETF.
	- Exclude any ETF where:
		Latest Close Price < 200-day SMA
		
3. Momentum Calculation (ROC)
	- Calculate Rate of Change (ROC) for each ETF.
	- ROC should be based on a configurable lookback period in months:
		1 month ≈ 21 trading days
		2 months ≈ 42 trading days
		etc. 
		Default lookback should be 1 month. If it is 1 month means it should use 21st rows if extracted data sorted in descending order. 
	- Formula:
		ROC (%) = ((Current Price / Price N days ago) - 1) * 100
4. Ranking
	- Rank all remaining ETFs based on ROC (descending).
	- Highest ROC = Rank 1.
